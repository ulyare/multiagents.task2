Описание алгоритма. На первом этапе агенты знают только своё число и свои связи с другими агентами. На втором этапе каждый агент передаёт всем доступным агнетам своё число. Каждый агент суммирует полученные значения со своим и делит их на количество полученных значений+1. Таким образом агент использует одну ячейку памяти. На третьем этапе происходят аналогичные действия. Критерием для остановки алгоритма было выбрано количество итераций (100 штук), так как на практике это оказалось лучшим решением. После прохода 100 шагов алгоритм считает считает реальное среднее арифметическое, благодаря чему можно сравнить отклонение.

В алгоритме также реализованы помехи (прибавление к передаваемому значению рандомного числа от -0.5 до 0.5) и отключение 0, 1 или 2 связей на каждом этапе с вероятностью 33%.

ИНСТРУКЦИЯ ПО ЗАПУСКУ!

1. В гитхабе нажать code и скопировать ссылку https.
2. Достать jade из архива или скачать jade.jar.
3. Открыть IDEA. File -> New -> Project from version control. Вставить ссылку и нажать clone.
4. Открыть File -> Project Structure -> Modules -> Dependencies. Нажать на плюсик и добавить JARs or directories. В окошке указать путь к jade.jar. Нажать ОК.
5. Можно запускать код.
